# 🏥 CareCore HMS – AI-Powered Hospital Management System

CareCore HMS is a production-grade, full-stack Hospital Management System designed to digitize and streamline hospital operations from patient registration to discharge, billing, and follow-up care. Built with modern web technologies, it supports multiple user roles with secure role-based access control and integrates AI-powered features to improve clinical decision-making and operational efficiency.

## ✨ Key Features

* 🔐 JWT Authentication with Role-Based Access Control (RBAC)
* 👨‍⚕️ Multi-role dashboards (Admin, Doctor, Patient, Receptionist, Nurse, Pharmacist, Lab Technician)
* 📅 Smart Appointment Scheduling & Queue Management
* 📋 Electronic Medical Records (EMR)
* 🩺 OPD & IPD Management
* 🛏️ Bed & Ward Management
* 💊 Pharmacy & Medicine Inventory
* 🧪 Laboratory Test & Report Management
* 💳 Billing, Payments & Invoice Generation
* 🔔 Real-time Notifications
* 📊 Analytics & Reporting Dashboard
* 🤖 AI-powered Clinical Intelligence

  * Symptom Pre-Triage Assistant
  * Medical History Summarizer
  * Appointment No-Show Prediction
  * Drug Interaction Detection
  * Intelligent Lab Result Analysis
